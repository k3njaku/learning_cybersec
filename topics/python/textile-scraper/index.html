The Mission // 🎯 What we're building

🧵 The Goal

Turn a website full of company listings into a clean spreadsheet of leads you can actually sell. Real gig. Real data.

📝 Imagine a vending machine...

Right now, textilepages.com is a vending machine full of snacks (leads). To get one, you have to put in a coin (click), grab the snack (copy the name), paste it in a spreadsheet, go back, repeat. For 1,000 snacks.

A scraper is a robot arm that does this 1,000 times per minute while you drink coffee.

Python is how we program the arm.

v1

Page 1 only

Scrape every company name + profile URL from the first page. Save to CSV. That's it.

v2

All pages

Follow pagination. Loop through every page until there are none left. Thousands of leads.

v3

Enrich the leads

Visit each company profile page, grab country, category, product images, contact hints.
