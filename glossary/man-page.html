Chapter 01 // Definition

📖 What's a man page?

Short for manual page. It's the official documentation for a Linux command, built into the operating system. Access it with man <command>.

📝 Think of it like IKEA instructions, but for commands

You bought a shelf. The box comes with a booklet that tells you every screw, every step, every warning. Linux commands work the same way — each ships with its own booklet. Except you don't have to dig through a drawer to find it; you just run man.

Every man page has a predictable structure: name, synopsis, description, options, examples, see also. Learn the layout once, read every man page faster.

Example: man ls

LS(1) User Commands LS(1) NAME ls - list directory contents SYNOPSIS ls [OPTION]... [FILE]... DESCRIPTION List information about the FILEs... -a, --all do not ignore entries starting with . -h, --human-readable with -l and -s, print sizes like 1K 234M 2G ...

Chapter 02 // Getting around

🎹 Man page keyboard shortcuts

Man pages open inside a pager (usually less). Once you learn these, you navigate like a pro.

Scrolling

  • ↓ / j — one line down
  • ↑ / k — one line up
  • Space / f — page down
  • b — page up
  • g — jump to top
  • G — jump to bottom

Searching & quitting

  • /pattern — search forward
  • ?pattern — search backward
  • n — next match
  • N — previous match
  • h — help (shows all keys)
  • q — quit (the key newbies forget 😅)
💡 Hot tip: The number in LS(1) isn't random — it's the manual section. Section 1 = user commands, 5 = file formats, 8 = system admin commands. When two things share a name (like passwd the command AND passwd the file), use man 5 passwd to pick the file format version.
why it matters
Chapter 03 // Why hackers love man pages

🎯 Recon AND exploit tool

🧪 Command discovery

You land a shell on a target. man -k (apropos) searches all man pages for a keyword. man -k network shows every tool related to networking. Instant recon of available tools.

🔍 GTFOBins pre-check

Found a binary with SUID? Check its man page for -e, --exec, or anything that runs a subprocess. Many privesc tricks (see GTFOBins) live in the man pages all along.

⚙️ Offline docs

No internet on the target? Man pages work offline. They're literally on the filesystem (/usr/share/man/).

🐚 Shell escape

Stuck in a restricted menu that lets you run man? Inside less, type !sh and you escape into a shell. Classic jailbreak trick.

Chapter 04 // Power moves

🛠️ Beyond man ls

# Search ALL man pages for a keyword man -k password # aka: apropos password # Jump to a specific section man 5 passwd # the /etc/passwd file format, not the command # Quick one-line summary whatis ls # ls (1) - list directory contents # See which sections have a page for 'printf' man -f printf # printf (1), printf (3) # Get JUST the synopsis man ls | head -20
🎓 Modern alternative: tldr gives you the 5 most common usage examples without wading through the full manual. Install with apt install tldr. Great when you just need "how do I tar this folder" without a 600-line deep dive.
