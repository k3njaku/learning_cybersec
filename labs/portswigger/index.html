Manifesto // Why no Burp here

🐢 Because Burp Community is SLOW

Burp Community is a rate-limited, Java-bloated trap designed to nudge you toward Burp Pro. For learning, it gets in the way.

📝 What you actually need to solve these labs

Every PortSwigger lab is just a web app. Web apps speak HTTP. Anything that speaks HTTP can attack them. You don't need a proxy with 47 tabs — you need:

  • A browser (you have one)
  • curl (already installed on Linux + macOS)
  • Your brain (the real star ⭐)
🚫 The "no Burp" rule: every step in every lab on this page can be done with the terminal or the browser. We use whichever is faster for that step. ADHD brains like linear, one-thing-at-a-time flows — that's exactly what a terminal gives you.
the arsenal
Chapter 01 // Your tiny, sharp toolkit

🔧 Three tools. That's it.

🌐 Browser URL bar

For any attack that lives in the URL (most SQLi, reflected XSS, path traversal). Paste, press Enter, read the response. Fastest feedback loop on Earth.

💻 curl in a terminal

For anything that needs POST requests, custom headers, cookies, or repeated fire. Reproducible, scriptable, zero GUI noise.

🦊 Firefox DevTools

Network tab → right-click a request → "Edit and Resend". It's a mini Repeater. Free. Already installed. No proxy cert nonsense.

☕ Real talk: if a lab genuinely needs a full proxy, we'll say so. But you'll be surprised how rare that is for Apprentice-level labs.

Chapter 03 // The shape of a lab page

📐 Every lab page, same structure

Consistency = less cognitive load. Every single lab in this vault has the same chapters in the same order:

01 · Refresher

Mini full-lesson on the vuln. Analogy, key facts, brain-power question. You can skip if you remember.

02 · Your Mission

The lab goal in plain English. What "solving" actually means.

03 · Tools & Setup

Exactly which tool you'll use (curl / URL bar / DevTools) and why.

04 · Guided Solve

Numbered steps. Each step = Do this → See this → Here's why.

05 · Payload Dissected

Character-by-character breakdown of the final injection.

06 · Real-world bounties

Real bug bounty reports using the same technique. Proof this isn't academic.

07 · Retention Quiz

1–2 MCQs. Get it right, confetti. Get it wrong, re-read.
